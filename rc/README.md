# rc/ — game-art resources

Generated branding/art for Villen: cover, banner, logo, icon. Produce these from
the brief in [`../docs/villen-art-brief.md`](../docs/villen-art-brief.md) and drop
the files here using the filenames listed in that brief's §5.

(Documentation diagrams live in `docs/`, not here — e.g. the architecture diagram
is `docs/villen-architecture.svg`. `rc/` is for game art.)
